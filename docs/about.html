Video Game Site Webring

As I was reading The SNES Omnibus by Brett Weiss I noticed that he included a lot of great game quotes from sites i'd never heard of. As an independent game site creator I realized that a tool to help discover independent and lesser known sites would be useful to the larger game community. In a couple days I hacked out a prototype and started collecting a list of game sites. I decided to model the base website on old 90's Webrings and Stumbleupon.

Video Game Site Webring is entirely Open Source and released under the MIT License.
